Canonical's solution for managing a fully integrated cloud-native infrastructure is named CNEP, the Cloud Native Execution Platform. This solution relies on the latest Canonical MAAS and Kubernetes releases. It provides hardening, auditing, confidential compute, encryption and identity management.

Get FIPS 140-2 encryption and a secure software foundation including a 10-year support lifespan with extensive security patching and compliance with NIST 800-53, PCI DSS, DISA STIG, ISO 27001, and NCSC TSA/TSR.

The optional Qualys agent provides critical security intelligence on demand and automates auditing, compliance and protection at the cluster level.
